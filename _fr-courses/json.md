[json](https://jsoneditoronline.org/) est principalement utilisé pour échanger des données (par exemple entre 2 programmes informatiques). En voici un exemple:
```json
{  
    "élèves":
    {  
        "nom":      "Dupont",   
        "prénom":   "Marcel",   
        "age":      12  
    },
    {  
        "nom":      "Durant",   
        "prénom":   "Gertrude",   
        "age":      14  
    }
}  
```
